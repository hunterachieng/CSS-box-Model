CSS Box Model

Question 1

A CSS box model is a box that wraps around every element in HTML.

Question 2

The following are the properties of the CSS box model:

  1. Content - This is the original content inside elements
  2. Padding - Creates space between content and element's border
  3. Border - Creates border around element
  4. Margin - Space between elements
  5. Width
  6. Height

Question 3

Calculating the total dimensions of a box given:
Width:400px
Margin : 20px
Padding: 20px
Border: 2px solid

Total Width = leftBorder + leftPadding + specifiedWidth + rightBorder + rightPadding
= 2px + 20px + 400px + 2px + 20px
= 444px

Total Height = topBorder + topPadding + specifiedLength + bottomBorder + bottomPadding
= 2px + 20px + 0px + 2px + 20px
= 44px
